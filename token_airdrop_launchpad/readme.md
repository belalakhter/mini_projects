## Usage:
- Update `./scripts/env.sh` with environment variables
- Run `npx hardhat clean`, `compile`, `typechain`, and `test`
- Deploy the contract to mainnet using `deploy.ts` in `scripts`
- Update the contract address in `./scripts/env.sh`
- Run `npx webpack` to build the frontend
